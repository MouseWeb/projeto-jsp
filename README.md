## Rodando o projeto

### Usando Maven + Spring Boot


Rodar o projeto
```shell
mvn spring-boot:run
```

[Acesse projeto](http://localhost:8080/)


Parar o projeto
```shell
mvn spring-boot:stop
```

[Acesse projeto](http://localhost:8080/)


### Usando Docker-compose

```shell
docker-compose -f docker/docker-compose.yaml up
```
